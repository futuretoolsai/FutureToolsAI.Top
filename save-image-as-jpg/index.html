Save Image As JPG

A simple Chrome extension that lets you convert any image on the web to JPG format with just a right-click.

Save Image As JPG is a lightweight Chrome extension designed to simplify the process of converting web images to the universally compatible JPG format. With a single right-click, you can convert any image format (WebP, AVIF, PNG, GIF, etc.) to JPG and save it to your computer.

No more dealing with incompatible image formats or using online conversion tools - this extension handles everything directly in your browser.

How to Use

1
Install the Extension
Add the Save Image As JPG extension to your Chrome browser from the Chrome Web Store.
2
Find an Image
Browse to any website with images you want to save as JPG. This works with any image format including WebP, AVIF, PNG, and more.
3
Right-Click the Image
Right-click on the image you want to convert. A context menu will appear with various options.
4
Select "Save Image As JPG"
Click on the "Save Image As JPG" option in the context menu. The extension will convert the image to JPG format immediately.
5
Save Your JPG File
Choose a location on your computer to save the JPG file. The file will be saved with the same name as the original image but with a .jpg extension.

Technical Details

The extension uses HTML5 Canvas technology to perform the image conversion. Here's how it works:

  1. The image is loaded into memory when you select it
  2. The image is drawn onto a canvas element
  3. The canvas content is converted to a JPG format (with 90% quality for optimal file size and image quality)
  4. The browser's download manager saves the converted image to your computer

This approach ensures that the conversion happens entirely on your local device, with no external services involved.

Troubleshooting

The "Save Image As JPG" option doesn't appear: Make sure the extension is properly installed and enabled in Chrome. You can check by going to chrome://extensions/ in your browser.

Can't convert some images: Due to web security restrictions (CORS - Cross-Origin Resource Sharing), some websites may prevent their images from being accessed by extensions. Unfortunately, this is a limitation of web security and not something the extension can bypass.

The converted image quality is too low/high: The extension uses a 90% quality setting for JPG conversion, which provides a good balance between file size and image quality for most uses.
